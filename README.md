# How to Use AI Sample Scout
1. Go to the Upload tab, then drag and drop folders containing .mp3 or .wav sample and one-shot files. The AI model reads a 1-second segment of each file.
2. Scroll down and index your samples with the AI model using the recommended batch size.
3. Go to the Search tab and start searching your sample library.

# Prompting Tips
1. Describe the role, not the tool. Avoid synth/plugin names and file formats.
❌ “Serum FM bass”
✅ “distorted bass hit”
2. Always include the sound's function. Use production roles people actually search for: kick, snare, clap, hi-hat, bass hit, bass stab, lead, pluck, pad, riser, impact, FX.
Example: “short punchy kick drum”
3. Add length and envelope. Use words like short, long, tight, sharp, sustained, or decaying.
Example: “short metallic percussion hit”
4. Describe character, not processing.
❌ “OTT compressed”
✅ “aggressive and punchy”
5. Use genre tags, but don't rely on them alone. Best results usually combine genre + role.
Examples: “techno kick drum”, “trap hi-hat one-shot”
6. Try multiple prompts per sound. This improves search accuracy.
Examples: “short distorted bass hit”, “aggressive bass stab”, “hard techno bass one-shot”
# Recommended Prompt Format
[length] + [character] + [role] + [genre]

“short punchy kick drum techno”
“long airy pad ambient”
“tight metallic hi-hat trap”
